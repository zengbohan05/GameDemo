# 斗地主游戏（Fight the Landlord）

> 一个基于 Java Swing 开发的单机版斗地主扑克游戏，支持三人对战（1 玩家 + 2 AI），完整实现发牌、叫地主、出牌、胜负判定等核心玩法。


## 🛠️ 技术栈

- **语言**：Java 8+
- **GUI 框架**：Swing
- **核心算法**：
  - 扑克牌洗牌与发牌逻辑
  - 叫地主随机决策（AI）
  - 出牌合法性校验（单张、对子、顺子、炸弹等）
  - 游戏流程状态管理
